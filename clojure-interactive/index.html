Why Clojure?

I like Clojure because:

  1. It has great support for interactive programming
  2. It has a great standard library
  3. It is a great tool for building your own abstractions.

Why Clojure 2

‚ÄúWe need to conform on languages‚Äù predicates that programming languages are like different breeds of horses. Different breeds of horses are mostly interchangeable.
